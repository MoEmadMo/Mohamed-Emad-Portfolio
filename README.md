## Featured project

### Laptop Price Prediction

An end-to-end machine learning application that predicts laptop prices using hardware specifications.

The workflow includes:

1. Collecting product data with BeautifulSoup and Selenium
2. Preparing and warehousing data with SQL Server and SSIS
3. Training the prediction model with Python and scikit-learn
4. Serving predictions through Flask
5. Exploring results through Power BI

[View the project on GitHub](https://github.com/MoEmadMo/Laptops_Price_Prediction)

## Run locally

Clone the repository and open `index.html`, or start a local static server:

```powershell
git clone https://github.com/MoEmadMo/Mohamed-Emad-Portfolio.git
cd Mohamed-Emad-Portfolio
python -m http.server 8000
```

Then visit `http://localhost:8000`.
