# Mohamed Emad Portfolio

A responsive, static portfolio site for Mohamed Emad, Data Engineer and BI Developer.

## Live publishing

This project is ready for GitHub Pages. After pushing it to a public GitHub repository:

1. Open the repository's **Settings**.
2. Select **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `master` branch and `/ (root)`, then save.

The public site will be available at:

`https://<github-username>.github.io/<repository-name>/`

## Run locally

Open `index.html` directly, or serve the folder with any static server:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
